## Bem-vindo ao repositório "Questionário Corintiano"

Este repositório contém um projeto desenvolvido durante o evento Next Level Week - Experts, organizado pela Rocketseat. O projeto consiste em um questionário sobre o time do Corinthians, uma aplicação web interativa construída com tecnologias modernas.

### Funcionalidades principais:
- **Questionário Dinâmico:** Os usuários podem responder a uma série de perguntas sobre o time do Corinthians.
- **Feedback Instantâneo:** Os resultados do questionário são exibidos instantaneamente aos usuários.
- **Design Responsivo:** A aplicação é totalmente responsiva, adaptando-se a diferentes dispositivos e tamanhos de tela.

### Tecnologias utilizadas:
- **HTML/CSS:** Para a estrutura e o estilo da aplicação.
- **JavaScript:** Para a lógica e a interatividade do questionário.
